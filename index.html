Description

LLM reasons in language, outputs text plans executed by separate skill policies.

Significance

First attempt to leverage LLM knowledge for robotics. Proved LLMs can reason about physical tasks but can't directly control robots.

Examples

SayCan, Code as Policies

Description

End-to-end transformer trained primarily on robot data. No VLM or video model backbone.

Significance

Proved a single policy could generalize across hundreds of tasks and environments. Laid groundwork for VLAs by demonstrating transformers work for robotics.

Examples

RT-1, Octo

Description

VLM backbone provides semantic understanding. Action decoder maps understanding → motion.

Significance

Dominant paradigm 2023-2025. First end-to-end approach inheriting web-scale knowledge from vision-language pretraining.

Examples

RT-2, OpenVLA, π0, GR00T N1

Description

Video backbone provides physical dynamics. IDM maps state transitions → actions.

Significance

Emerging paradigm. 10x+ data efficiency over VLAs in early results by inheriting physics understanding from video.

Examples

Cosmos Policy, mimic-video, 1XWM

Description

Learns to predict future states from video. Doesn't output actions directly.

Significance

Foundation for VAMs. Enables planning by "imagining" outcomes before acting.

Examples
